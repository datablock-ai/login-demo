# Simple Web3 Login

This is a simple Web3 login that allows you to connect to your wallet, and to use
use your wallet to sign a message (a mechanism often used to log in in Web3 apps).

![app_preview.png](public%2Fapp_preview.png)

## Install

### Yarn

`yarn install`

### npm

`npm install`

## Run

### Yarn

`yarn dev`

### npm

`npm run dev`
